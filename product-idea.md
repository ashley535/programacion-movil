# Entregable - Semana 1

## Ficha de Idea de Producto

### Nombre tentativo de la app
StudyMobile

### Problema que resuelve
Muchos estudiantes tienen dificultades para organizar tareas, exámenes y proyectos académicos, lo que provoca retrasos y bajo rendimiento.

### Usuario objetivo (perfil)
Estudiantes universitarios de entre 18 y 30 años que desean mejorar su organización académica.

### Jobs-to-be-done
1. Organizar tareas y proyectos académicos.
2. Recibir recordatorios de fechas importantes.
3. Dar seguimiento al progreso de sus asignaturas.

### Propuesta de valor
"Una aplicación móvil que ayuda a los estudiantes a gestionar sus actividades académicas de forma simple y eficiente."

### Tipo de app
Productividad / Educación

---

## Backlog Inicial

### Prioridad Alta

- Como estudiante, quiero crear tareas para organizar mis actividades.
- Como estudiante, quiero establecer fechas límite para recordar entregas importantes.
- Como estudiante, quiero recibir notificaciones para no olvidar exámenes o tareas.
- Como estudiante, quiero marcar tareas como completadas para visualizar mi progreso.

### Prioridad Media

- Como estudiante, quiero clasificar tareas por asignatura para mantener el orden.
- Como estudiante, quiero consultar un calendario para ver mis actividades programadas.
- Como estudiante, quiero editar tareas para actualizar información cuando sea necesario.

### Prioridad Baja

- Como estudiante, quiero personalizar mi perfil para adaptar la aplicación a mis preferencias.
- Como estudiante, quiero visualizar estadísticas de tareas completadas para medir mi desempeño.
- Como estudiante, quiero utilizar modo oscuro para mejorar la experiencia visual.

## Prompt utilizado

Genera una ficha de idea de producto para una aplicación móvil dirigida a estudiantes universitarios, incluyendo problema que resuelve, usuario objetivo, jobs-to-be-done, propuesta de valor y backlog inicial con historias de usuario clasificadas por prioridad.
